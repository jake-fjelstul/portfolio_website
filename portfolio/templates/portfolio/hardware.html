Desk Assistant

I designed and prototyped a modular Desk Assistant using Arduino hardware and custom 3D-printed components to enhance workspace productivity. The system features an LCD menu interface for tracking focus timers and break intervals , integrated with a robotic arm that utilizes ultrasonic sensors to enable gesture-controlled lighting. I housed these electronics in a custom enclosure with a built-in wire management base. The project was designed in two phases so I am currently working on designing the system to run on a single unified microcontroller.
